# 《共识协议》(Consensus Protocol)
 **[简介]**：这是一款基于 Unity 开发的融合了潜行探索、身份伪装与卡牌辩论的微动作/剧情推理游戏。玩家将步入深渊，记忆重演角色，在错综复杂的设施中收集记忆碎片与证据，最终在惊心动魄的庭审中，利用手中的证据卡牌引导众人，达成最终的“共识”。

## 💻 运行环境与技术栈

* **游戏引擎**：Unity [6000.3.6f1 LTS]
* **目标平台**：Windows (PC)
* **输入设备**：键盘 + 鼠标 即可

## 🚀 如何运行体验

**选项 A：直接运行打包好的游戏**
1. 进入本仓库的release部分。
2. 找到最新版本（当前为v0.1.0）下载`[consensus protocol.zip]`后运行`[测试版本.exe]`。

**选项 B：在 Unity 编辑器中打开（适合查看源码）**
1. 确保您的电脑已安装 Unity `[6000.3.6f1]`。
2. 在release最新版本中下载源码。
3. 在 Unity Hub 中点击 "Add project from disk"，选择源码所在的根目录。
4. 打开项目后，进入 `Assets/Scenes` 目录，双击打开 `[初始场景MainMenu]` 场景。
5. 点击编辑器上方的 Play (▶) 按钮即可试玩。

## 🕹️ 怎么玩 (操作说明)

### 基础操作
* `W` `A` `S` `D`/`鼠标移动`：移动角色 / 控制视角
* `鼠标左键`：互动 / 确认 / 小游戏游玩
* `ESC`/`E`：暂停游戏 / 呼出菜单
* 详细操作请见游戏内设定

### 核心玩法流程（现阶段，未来计划扩展至更多角色，开发不同章节和更多小游戏）
开局可选择是否接入API自行设定卡牌内容，NPC证词等
未来扩展至多议题，精细化混乱值系统
目前第一章游戏围绕**“潜渊 - 伪装 - 庭审”**三大阶段展开：

1. **第一阶段：潜入深渊 (收集与潜行)**
   * 游戏开场将发布本次审判的“议题”。玩家需要收集散落的**“记忆碎片”*，并潜入深渊走廊，巧妙躲避安保人员的巡逻视线。
   * 成功突破封锁即可获得第一张【证据牌】（塔罗牌形式）。

2. **第二阶段：身份伪装 (解谜与获取证据)**
   * 玩家需要选择不同的伪装身份潜入核心区域，每个身份对应不同的挑战，成功即可获得关键【证据牌】：
     * **清洁工**：进行“接水管”解谜小游戏，修复管道系统。
     * **技术人员**：面对NPC的严厉盘问，谨慎选择对话选项以蒙混过关。
     * **“罪徒”原身份**：潜入终端拷贝机密数据，完成特定的破解小游戏。

3. **第三阶段：终局庭审 (卡牌辩论)**
   * 带着收集到的【证据牌】进入最终庭审现场。
   * 玩家需要仔细聆听NPC的证词，并观察他们的**隐藏属性**。
   * **制胜关键**：根据局势，在正确的时机打出对应的【证据牌】来反驳或引导NPC，最终让所有人达成一致，完成“共识”判决！

## ✨ 游戏特色 (Highlights)

* **高自由度的自建剧情系统 (API 接入)**：我们为游戏底层开发了一套灵活的剧情解析器。游戏不仅包含内置的主线流程，还**支持通过接入 API等来调整输出风格，动态生成“卡牌内容”、“证词”和对玩家输入的证词进行评分**。这意味着《共识协议》不仅仅是一个固定流程的游戏，更是一个**可扩展的剧情推理引擎**，玩家或创作者可以完全自定义自己的剧本！
* **丰富的复合玩法**：将紧张的潜行躲避、多样的解谜小游戏（如接水管、颤栗迷宫）与策略卡牌玩法巧妙结合，拒绝单一枯燥的游戏体验。
* **深度的庭审博弈**：独创的“共识”卡牌系统，玩家需要像真正的律师/调查员一样，洞察NPC的属性与证词漏洞，打出致命的证据牌。玩家的每一次选择都会影响最终的“共识”
* **沉浸式的悬疑叙事**：“议题”先行，悬念拉满。每一次伪装和潜行，都是为了揭开“罪徒”背后的真相。
